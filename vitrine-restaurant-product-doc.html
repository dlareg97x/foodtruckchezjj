Contexte & Objectifs

Cadrage projet — client, cible, contraintes

⚡ Piloton intégré dès le MVP

Ce site vitrine n'est pas un simple site de présentation. Il est conçu comme un outil de génération d'avis Google via le funnel Piloton. Le QR code plaquette → landing satisfaction → redirection Google est intégré nativement dans la structure du site, pas ajouté en option.

🎯 Cible client
Ouvriers, artisans, employés zone industrielle, étudiants. Usage mobile quasi-exclusif. Décision d'achat en moins de 30 secondes.
Implication : Design ultra-rapide, CTA immédiat
📍 Zone géographique
Local first. SEO ciblé sur ville + quartier + zone industrielle. Google Maps omniprésent.
Implication : Schema LocalBusiness obligatoire
⏱️ Fenêtre de décision
Client du midi avec 45 min de pause. Il décide en 20s max. Le site doit convertir immédiatement.
Implication : LCP < 1s, bouton tel above-the-fold
⭐ Réputation Google
Les avis Google = levier principal de nouveaux clients. Piloton automatise la réponse et optimise le funnel de collecte.
Implication : QR code + landing Piloton = core feature

Modules UX

Toutes les fonctionnalités d'expérience utilisateur

Mobile-first absolu. L'utilisateur est debout, au soleil, avec les doigts sur smartphone. Chaque interaction doit être faisable au pouce, en moins de 2 taps.

Impact : +30-50% de clics téléphone

Impact : Conversion visiteur → client physique

Affichage "OUVERT" / "FERMÉ" calculé en temps réel selon les horaires configurés. Badge vert/rouge visible dès la home.
Impact : Réduction des appels pour horaires
📌 Sticky Header Mobile
Header fixe avec logo + téléphone + burger menu. Reste accessible en scrollant.
Impact : UX fluide, accès permanent aux CTA
🌙 Dark Mode
Basculement auto selon préférence système + toggle manuel. Confort pour usage en extérieur et nuit.
Impact : Accessibilité + modernité
📶 Mode Offline (PWA)
Le menu et les infos de contact restent accessibles sans connexion (Service Worker + cache).
Impact : Zone industrielle = connexion instable
🔔 Notification Push (PWA)
Opt-in notifications : nouvelle promo, plat du jour, horaires exceptionnels.
Impact : Canal de réengagement direct
📱 "Installer l'app" (PWA)
Prompt d'installation sur l'écran d'accueil. Icône branded. Expérience app sans store.
Impact : Fidélisation, retour client
🎞️ Menu Filtrable / Catégories
Onglets Tacos / Sandwichs / Boissons / Desserts. Filtre allergènes. Recherche rapide.
Impact : Temps de choix réduit, UX fluidifiée

Section "Aujourd'hui" mise à jour manuellement ou programmée. Affiché en hero sur mobile.

Galerie swipeable, zoom pinch-to-zoom, navigation gesture. Photos optimisées WebP.

Apparition au scroll, hover states, feedback tactile, skeletons de chargement.

Fonctionnalités Marketing

Tous les outils de croissance et de fidélisation

⚡ Intégration Piloton — Funnel Avis Natif

Le site intègre le funnel complet de collecte d'avis Piloton : le client scanne le QR code (plaque plexiglas branding restaurant + Piloton), arrive sur /avis/satisfaction, choisit sa note. Si ≥ 4 étoiles → redirection directe vers Google Reviews. Si < 4 étoiles → formulaire privé capturé dans le dashboard Piloton. Les réponses aux avis Google sont générées automatiquement par Claude AI (3 variantes de ton). Le restaurant reçoit une notif admin pour chaque nouvel avis.

🔲 QR Code Funnel Avis PILOTON
QR code sur plaque plexiglas double-branding → landing satisfaction → filtre Google vs. feedback privé.
Impact : +200% avis Google en 3 mois
🤖 Réponses Auto Avis PILOTON
Claude AI génère 3 variantes de réponse par avis. Le gérant choisit ou valide. Ton personnalisé.
Impact : Gain de temps + note Google améliorée
📊 Dashboard Réputation PILOTON
Vue consolidée des avis, évolution de la note, avis non répondus, feedbacks privés.
Impact : Pilotage de l'e-réputation en temps réel
📧 Capture Email
Popup opt-in post-commande ou footer. Offre de bienvenue (coupon -10% sur prochain achat).
Impact : Construction liste marketing owned
💬 WhatsApp Business
Bouton WhatsApp flottant. Lien vers catalogue produit WhatsApp Business. Liste de diffusion promos.
Impact : Canal préféré des ouvriers/artisans
🎟️ Coupon Digital
Coupon unique généré à l'inscription email. Code à montrer à la caisse. Tracking de l'usage.
Impact : Incite la 1ère visite
🏆 Programme Fidélité Digital
Carte de fidélité digitale. 10 tampons = 1 repas offert. Via email ou QR code client.
Impact : Rétention client long terme
👥 Parrainage
Lien unique de parrainage. Récompense pour le parrain + le filleul à la 1ère commande.
Impact : Acquisition virale à coût quasi-nul
🔔 Popup Promotion
Popup temporisé (après 10s) ou au scroll 50%. Offre du moment. Fermeture facile (UX non intrusive).
Impact : +15% de conversion sur promo en cours
📱 SMS Marketing
Collecte numéro opt-in. Envoi promo spéciale midi le lundi matin. Via Twilio ou Sendinblue.
Impact : Taux d'ouverture SMS : 98%
📰 Newsletter Hebdo
Email hebdo : plats de la semaine, promo, news. Génération semi-auto via IA.
Impact : Fidélisation et retour régulier
🔥 Heatmap Comportemental
Intégration Hotjar/Microsoft Clarity. Voir où les utilisateurs cliquent, scrollent, abandonnent.
Impact : Optimisation continue du site

Performance Technique

Le site doit charger en moins d'1 seconde sur 4G

🖼️ Images WebP + AVIF
Conversion automatique des photos en WebP/AVIF. Réduction poids images -60% sans perte de qualité visible.
Impact : LCP réduit de 40%
🦥 Lazy Loading
Chargement différé des images hors viewport. Native HTML + Intersection Observer pour les composants JS.
Impact : First Load -50% de données
⚡ CDN Global
Assets statiques servis depuis Cloudflare CDN. Latence minimale quelle que soit la connexion.
Impact : TTFB < 100ms
📦 Minification CSS/JS
Bundling, tree-shaking, minification, code splitting. Taille des fichiers réduite au minimum.
Impact : Bundle -40%
📱 PWA complète
Service Worker, App Manifest, cache offline, add-to-homescreen. Expérience app native sans store.
Impact : Engagement +2x, offline garanti
💾 Cache Navigateur
Headers Cache-Control optimisés. Assets statiques mis en cache 30 jours. HTML : no-cache pour fraîcheur.
Impact : 2ème visite instantanée
🔤 Font Subsetting
Ne charger que les caractères latins nécessaires. Font-display:swap pour éviter FOIT.
Impact : Fonts -70% de poids
📊 PageSpeed Score 90+
Audit Lighthouse automatisé en CI/CD. Score > 90 en mobile comme en desktop. Non-négociable.
Impact : SEO + UX + conversion

Sécurité

Protection de base non-négociable

🔒 HTTPS Forcé
SSL/TLS obligatoire. Redirection HTTP → HTTPS. HSTS header. Let's Encrypt auto-renew.
Impact : Obligatoire — facteur SEO
🤖 Anti-Spam Formulaires
Honeypot fields, rate limiting, reCAPTCHA v3 invisible. Bloquer les bots sans friction UX.
Impact : Données de contact propres
⏱️ Rate Limiting API
Limiter les appels API (avis, formulaires, auth). Nginx / Cloudflare rate limit. Protège contre DDoS simples.
Impact : Stabilité serveur
🛡️ Headers Sécurité
CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy. Configuration Cloudflare.
Impact : Protection XSS/clickjacking
🔐 Auth Admin Sécurisée
JWT tokens, session expiry 8h, 2FA optionnel, logs de connexion, blocage après 5 échecs.
Impact : Protège l'accès back-office
💾 Backup Quotidien
Sauvegarde DB + fichiers chaque nuit. Retention 30 jours. Restore en 1 commande.
Impact : Continuité d'activité
🔏 RGPD Compliance
Consentement granulaire, droit à l'effacement, registre des traitements, DPA avec sous-traitants.
Impact : Légal obligatoire en France
🔍 Monitoring Uptime
UptimeRobot / Better Uptime. Alerte SMS si site down > 1 min.
Impact : Réactivité en cas de panne
